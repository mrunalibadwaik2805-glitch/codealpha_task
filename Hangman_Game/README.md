# Task 1 - Hangman Game

## Description
This is a simple text-based Hangman Game developed in Python.

## Features
- Randomly selects one word from a list of five words.
- Allows six incorrect guesses.
- Displays the hidden word using underscores.
- Updates the word after each correct guess.
- Console-based application.

## Technologies Used
- Python 3

## Concepts Used
- Random module
- While loop
- If-Else statements
- Strings
- Lists
